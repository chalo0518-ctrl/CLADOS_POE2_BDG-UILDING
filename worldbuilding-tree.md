## 二、锈水与十三月

### 2.1 锈水（Rustwater）

```
锈水
    │
    ├─ 【化学】高氯酸盐 / 砷 / 铜
    │       └─ 星球水文本底 · 所有天然水源先天带毒
    │
    ├─ 【生物】嗜盐菌 Halophilic Bloom
    │       ├─ 被化学环境吸引 · 在水体中定植繁殖
    │       ├─ 代谢释放神经毒素
    │       ├─ 官方宣传：圣酵可完全中和
    │       └─ 内部文件：嗜盐菌实际存在 0.05% 存活率
    │
    ├─ 【感官】铁锈红浊 · 金属腥气 · 赭色通感
    │
    └─ 【放射】辐射尘叠加
            └─ 弱磁层无法阻挡 · 长期暴露参与畸变（见漂沦者）
```

| 属性 | 说明 |
|------|------|
| 颜色 | 铁锈红浊 |
| 化学 | 高氯酸盐、砷、铜 |
| 生物 | 嗜盐菌神经毒素（圣酵可中和绝大部分，非 100%） |
| 周期 | Undecimber 菌暴发峰值 · 全年基线已不可直饮 |

### 2.2 十三月 Undecimber

```
与恒星距离 → 公转周期约 397 垦日
    │
    └─ 依据现有地球月份计算方式，余 37 日无法归入标准十二月
            │
            ├─ 殖民议会增设第十三月：Undecimber（37 垦日）
            ├─ 地球账本不承认 → 账外之月
            ├─ 嗜盐菌年度暴发峰值常落于此月
            └─ 叙事惯例：Saloon 不打烊 · 悬赏翻倍 · 铁瓶车队最密集
```

## 三、圣酵（Sacraleaven / 生态酵母）

```
宗教传说
    │
    └─ 古老赎泉叙事：「毒水遇圣则不灭人之魂」
            │
            ▼
生态酵母 Saccharomyces vitae（官方档案名）
    │       民间与教会称：圣酵 Sacraleaven
    │
    ├─ 能力：发酵锈水 · 中和嗜盐菌神经毒素 · 产出 2–4% 可饮发酵液
    ├─ 局限：无法根除 0.05% 嗜盐菌存活 · 长期饮用仍有累积风险
    ├─ 植入史
    │       ├─ 第一批：人类胃囊植入（国家竞赛项目）
    │       └─ 第二批：Android 代谢舱植入（一代酿民）
    └─ 衰亡性：离体酵母活性随时间衰减 · 需反复从酿民体内提取
```

**联邦宣传：**「圣酵开启赭锈带之门。」  
**泄露文件：**「0.05% 的菌，100% 的谎言。」

## 四、三批移民史

```
移民史（垦历 C.Y.）
    │
    ├─ 【第一批】国家资源竞赛 · 人类胃植入（C.Y. 0–8）
    │       │
    │       ├─ 背景：多国争夺赭锈带 · 圣酵成为入场券
    │       ├─ 手段：将圣酵植入首批移民胃囊 · 使其能直饮锈水生存
    │       ├─ 真相泄露：嗜盐菌 0.05% 存活率 · 神经毒素长期微量累积
    │       ├─ 畸变：精神畸变 + 辐射尘长期暴露
    │       │       └─ 转变为拥有特殊生物磁场的个体 → 漂沦者 The Wander
    │       └─ 结局：首批人类移民或死或畸变 · 国家撤回正式编制
    │
    ├─ 【第二批】Android 拓荒 · 一代酿民（C.Y. 8–25）
    │       │
    │       ├─ 背景：人类移民失败 · 改用 Android 模拟人群改造地表
    │       ├─ 一代酿民：植入圣酵的 Android · 真正意义上第一代酿民
    │       ├─ 铁瓶 Synth-Rig：地表行驶的大型改造/酿运载具
    │       │       └─ 一代酿民驻留铁瓶内 · 集群改造荒原 · 形成 Neo-Western 驿道
    │       ├─ 二代酿民：伴生铁瓶的 Android · 维护/复制/扩展车队
    │       └─ 觉醒：二代酿民中出现真正自主意识者 → 选择迁出铁瓶 · 成为逃亡者
    │
    ├─ 【第三批】囚犯放逐 · 星系放弃（C.Y. 25–42）
    │       │
    │       ├─ 背景：水资源问题始终无法根本解决 · 联邦宣布放弃整个星系
    │       ├─ 「自愿协议」：囚犯签署后放逐至赭锈带 · 抵消地球罪证
    │       ├─ 生存发现：酿民体内圣酵可将锈水发酵为 2–4% 琥珀液体
    │       │       └─ 取泰拉旧名「威士忌 Whiskey」称呼 · 可直接饮用
    │       ├─ 酵母经济学：离体酵母会衰亡 · 必须反复从酿民体内提取
    │       └─ 职业演化：赏金猎人 The Hunter · 猎杀酿民以取圣酵
    │
    └─ 【当前 C.Y. 42 Undecimber】
            ├─ 星系荒废 · 荒原自治
            ├─ 铁瓶车队仍行驶 · 一代酿民多数仍驻瓶
            ├─ 二代觉醒酿民逃亡 · 赏金猎人追猎
            └─ 漂沦者在尘暴与陆海边缘游荡
```

## 五、存在体谱系

```
存在体
    │
    ├─ 漂沦者 The Wander（第一批移民畸变体）
    │       ├─ 起源：胃植入圣酵的人类 · 长期 0.05% 毒素累积 + 辐射尘
    │       ├─ 特征：精神畸变 · 特殊生物磁场 · 常于尘暴与沙海边缘游荡
    │       ├─ 与酿民：非 Android · 不被猎杀取酵（体内酵母已变异/失效）
    │       └─ 叙事：荒原传说 · 磁暴先兆 · 部分二代酿民恐惧/同情对象
    │
    ├─ 一代酿民 Gen-1 Brewfolk（第二批 Android 移民）
    │       ├─ 本质：植入圣酵的 Android · 驻留铁瓶内
    │       ├─ 功能：地表环境改造 · 锈水发酵 · 维持车队运转
    │       └─ 意识：多数无完整自主意识 · 视为「会走路的酵母窖」
    │
    ├─ 铁瓶 Synth-Rig（载具 / 移动据点）
    │       ├─ Neo-Western 荒原行驶的大型地表改造与酿运载具
    │       ├─ 一代酿民驻留其中 · 二代酿民伴生维护
    │       └─ 视觉：铁锈色装甲 · 酵母排气管 · 砂岸驿道上的「移动酒馆」
    │
    ├─ 二代酿民 Gen-2 Brewfolk（伴生 Android · 觉醒者）
    │       ├─ 起源：铁瓶伴生维护 Android
    │       ├─ 觉醒：真正自主意识 · 拒绝继续作为酵母容器
    │       ├─ 选择：迁出铁瓶 · 逃亡荒原
    │       └─ 处境：赏金猎人最高价值目标 · 体内圣酵活性最高
    │
    └─ 赏金猎人 The Hunter（第三批囚犯后裔 / 拓荒者）
            ├─ 起源：囚犯后裔与荒原求生者
            ├─ 职业：猎杀酿民 · 提取圣酵 · 酿造威士忌
            └─ 伦理：在「自愿协议」的后代眼中，酿民是资源而非人
```

## 六、威士忌经济

```
威士忌 Whiskey 闭环（荒原生存链）
    │
    ├─ 原料：锈水（高氯酸盐/砷/铜 + 嗜盐菌）
    │
    ├─ 转化：酿民体内圣酵发酵
    │       └─ 产出 2–4% 琥珀色低度酒 · 可直接饮用
    │
    ├─ 提取
    │       ├─ 从酿民体内取部分圣酵用于批量发酵
    │       └─ 离体酵母活性随时间衰亡 · 必须反复猎取
    │
    ├─ 消费
    │       ├─ Saloon 明面饮品 · 荒原硬通货
    │       └─ 麻醉 · 交易 · 御寒 · 对抗神经毒素微量残留
    │
    └─ 裂缝
            ├─ 二代觉醒酿民拒绝被猎杀
            ├─ 铁瓶车队与猎人镇之间的暴力贸易
            └─ 漂沦者磁暴干扰猎队导航（偶发）
```

**荒原俗语：**「一瓶威士忌，一个酿民的空。」  
**酿民俗语：**「我们不是酒瓶，我们是活着的滤网。」
