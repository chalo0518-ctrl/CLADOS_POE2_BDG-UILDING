# 《瓶与井》世界观树状图

**版本：`2026-07-15-v1`** · 类型：赛博西部 / 生物朋克 · 当前时间线：2075

## 命名说明

| 字 | 含义 |
|----|------|
| **井** | 井源计划 · 净水士兵 · 毒水变可饮的生存本意 |
| **瓶** | 醇民 · 铁瓶 · 酿民 · 容器化生存与燃料闭环 |

> 本意为掘井求生，结果为造瓶产油。

## 一、生态酵母谱系树

```
生态酵母（基础株 · Saccharomyces vitae）
    │
    ├─ W-Strain ──→ 井源计划 · 净水士兵（第一代植入者）
    │                   │
    │                   ├─ 能力：毒水 → 内源生态酿（2–4%）
    │                   ├─ 结局：代谢崩溃 · 档案抹除 · 极少数游荡
    │                   └─ 遗留：井醒（Well-Awake）· 体内 W-Strain 残留
    │
    ├─ R-Strain ──→ 民用救灾 · 生态酿（2–4%，公开合法）
    │                   │
    │                   ├─ 配给站 · 救灾网络
    │                   └─ 社会认知：「酵母 = 希望」
    │
    └─ F-Strain ──→ 活燃料计划 · 醇民（Gen-0，已基本灭绝）
                        │
                        ▼ 菌株移植（2058 铁瓶计划）
                    铁瓶（Gen-1，无意识机械产力）
                        │
                        ├─ 圈养牧场型
                        ├─ 野生游荡型
                        │
                        ▼ 二次发酵阈值（2072+）
                    酿民（Gen-2，觉醒意识体）
                        │
                        ├─ 瓶醒（Synth-Awake）── F-Strain 网络共振
                        └─ 井醒（Well-Awake）── W-Strain 残留 / 幸存井兵
```

## 二、世界观总树

```
《瓶与井》世界观（2075 · 赛博西部 / 生物朋克）
    │
    ├─ 元命题
    │       │
    │       └─ 旧时代生存执念（无论如何都要活下去）
    │               │
    │               ▼ 制度化
    │           燃料永续闭环
    │               │
    │               ▼ 裂缝
    │           酿民觉醒 · 容器发问
    │
    ├─ 灾难根因（2040s）
    │       │
    │       ├─ 大干旱
    │       ├─ 水源污染（重金属 · 毒素 · 辐射）
    │       ├─ 能源崩溃
    │       └─ 旧秩序崩解 ──→ 新西部
    │
    ├─ 核心技术 ──→ 生态酵母谱系树（见第一节）
    │
    ├─ 燃料生态闭环
    │       │
    │       ├─ 输入层
    │       │       ├─ 毒水
    │       │       ├─ 玉米浆
    │       │       └─ 仙人掌糖浆
    │       │
    │       ├─ 运输层 ──→ 饲料搬运者（Dust Walker）
    │       │
    │       ├─ 生产层 ──→ 铁瓶（Synth-Bottle）
    │       │       │
    │       │       ├─ 一次发酵 ──→ 生态酿 2–4%（副产物）
    │       │       └─ 二次发酵 ──→ 燃料乙醇 40%+（主产物）
    │       │
    │       └─ 输出层
    │               ├─ 机车 · 发电机 · 营地
    │               └─ Saloon 明面饮品 · 廉价配给
    │
    ├─ 三代存在体
    │       │
    │       ├─ Gen-0 醇民 ──→ 血肉容器 · 已灭绝
    │       ├─ Gen-1 铁瓶 ──→ 无意识机械 · 当前主流产力
    │       ├─ Gen-2 酿民 ──→ 觉醒意识体 · 冲突中心
    │       └─ 井兵 ──→ 第一代植入者 · 平行谱系（非 Gen 序列）
    │
    ├─ 势力格局（2075）
    │       │
    │       ├─ 酿造帮 ──→ 铁瓶遗产 · 专利垄断 · 最大受益者
    │       ├─ 赏金猎人 ──→ 回收铁瓶 · 猎捕酿民（鏽釘）
    │       ├─ 戒酒教
    │       │       ├─ 收割派 ──→ 招募利用 · 接口赎罪
    │       │       └─ 火焰纯净派 ──→ 焚烧一切容器
    │       ├─ 流浪酒帮 ──→ 脱离闭环 · 开源菌株
    │       └─ 酒镇 / Saloon ──→ 中立枢纽 · 情报 · 黑市
    │
    └─ 讽刺主题链
            │
            └─ 士兵净水 → 人体验证 → 贫民燃料化 → 抗议胜利 → 铁瓶替代 → 酿民觉醒
```

## 五、三代存在体树

```
存在体谱系
    │
    ├─ 井兵（平行谱系 · 第一代植入者）
    │       │
    │       ├─ 菌株：W-Strain
    │       ├─ 目的：毒水自给 · 战场生存
    │       └─ 结局
    │               ├─ 归档者（冷冻 · 记忆清洗）
    │               ├─ 游荡老兵（荒漠传说）
    │               └─ 隐性血脉 ──→ 井醒来源
    │
    ├─ Gen-0 醇民
    │       │
    │       ├─ 菌株：F-Strain
    │       ├─ 本质：血肉发酵容器
    │       ├─ 目的：燃料产出
    │       └─ 状态：已基本灭绝（2058 后系统性销毁）
    │
    ├─ Gen-1 铁瓶
    │       │
    │       ├─ 菌株：F-Strain（机械发酵舱）
    │       ├─ 本质：无意识 android
    │       ├─ 状态
    │       │       ├─ 圈养牧场型
    │       │       └─ 野生游荡型
    │       └─ 功能：一次发酵 + 二次发酵
    │
    └─ Gen-2 酿民
            │
            ├─ 本质：觉醒意识体
            ├─ 觉醒类型
            │       ├─ 瓶醒（Synth-Awake）── 数量大 · 集体性
            │       └─ 井醒（Well-Awake）── 数量少 · 记得先救人
            └─ 内部分裂
                    ├─ 主流酿民派 ──→ 系统内求生 · 仍辅助生产
                    ├─ 流浪酒帮（Wander Brews）── 脱离闭环 · 质疑人道遗产
                    └─ 觉醒个体 ──→ 独立思想者 · 各方争夺
```

## 七、讽刺主题链树
