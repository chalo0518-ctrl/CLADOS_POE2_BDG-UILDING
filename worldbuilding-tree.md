```

## 二、锈水危机

### 2.1 三层污染（官方主干 A + B + 辐射尘）

```
锈水来源（官方主干）
    │
    ├─ 【化学层】方案 A · 深井毒卤 ── 全年基线
    │       └─ 深井穿透卤层 → 高氯酸盐 / 砷 / 铜
    │
    ├─ 【生物层】方案 B · 流浪赤潮 ── Undecimber 峰值
    │       └─ 嗜盐菌群 → 神经毒素 · 水呈铁锈红浊
    │
    └─ 【放射层】辐射尘 ── 尘暴季 / 沙海扬尘
            ├─ 弱磁层无法阻挡双恒星风与轨道衰变尘
            ├─ 辐射尘沉降 → 渗入重砂层 → 随卤水上涌进入深井
            ├─ 沙海翻涌与沙豚集群迁徙 → 扬尘脉冲式污染
            └─ 三层叠加 → 水质降至「不可接触」级
```

| 层级 | 方案 | 污染机制 | 周期 |
|------|------|----------|------|
| 化学 | **A 深井毒卤** | 殖民深井穿透地下卤层，采出水含重金属盐 | 全年基线 |
| 生物 | **B 流浪赤潮** | 嗜盐菌群载入水源，代谢产神经毒素 | **Undecimber 高峰** |
| 放射 | **辐射尘** | 放射性微粒随尘暴与沙海扬尘进入水源 | 尘暴季 + Undecimber 叠加 |

> **日常：** 化学卤毒已令水不可直饮。  
> **Undecimber：** 赤潮 + 辐射尘扬尘峰值叠加，全年最毒月份。  
> **生态酵母：** 唯一可规模化的三重净化应答（见第三节）。

## 三、技术应答

### 3.1 生态酵母谱系

```
生态酵母（基础株 · Saccharomyces vitae）
    │       三重净化：化学毒 + 生物毒 + 辐射尘 → 酵母泥需专门处置
    │
    ├─ W-Strain ──→ 井源计划 · 净水士兵（井兵）
    │                   ├─ 锈水 → 内源生态酿（2–4%）
    │                   ├─ 结局：代谢崩溃 · 档案抹除 · 极少数游荡
    │                   └─ 遗留：井醒（Well-Awake）· W-Strain 残留
    │
    ├─ R-Strain ──→ 民用救灾 · 生态酿（2–4%，公开合法）
    │                   ├─ 配给站 · 救灾网络
    │                   └─ 社会认知：「酵母 = 希望」
    │
    └─ F-Strain ──→ 活燃料计划 · 醇民（Gen-0，已基本灭绝）
                        │
                        ▼ 菌株移植（C.Y. 23 铁瓶计划）
                    铁瓶（Gen-1，无意识机械产力）
                        │
                        ├─ 圈养牧场型 / 野生游荡型
                        │
                        ▼ 二次发酵阈值（C.Y. 37+）
                    酿民（Gen-2，觉醒意识体）
                        │
                        ├─ 瓶醒（Synth-Awake）── F-Strain 网络共振
                        └─ 井醒（Well-Awake）── W-Strain 残留 / 幸存井兵
```

**殖民宣传语：**「锈水变活路。」（刻意不提颜色与气味）  
**回收帮私下：**「酵母泥比锈水更麻烦。」

### 3.2 燃料永续闭环

```
燃料永续闭环（铁瓶计划遗产 · C.Y. 23–）
    │
    ├─ 输入层
    │       ├─ 锈水（锈卤 + 赤潮 + 辐射尘）
    │       ├─ 玉米浆
    │       └─ 仙人掌糖浆（固陆作物 · 砂岸种植带）
    │
    ├─ 地理层 ──→ 陆海重砂 · 辐射尘富集 · 野生铁瓶漂移带
    │
    ├─ 运输层 ──→ 饲料搬运者（Dust Walker · 砂岸驿道）
    │
    ├─ 生产层 ──→ 铁瓶（Gen-1 · Synth-Bottle）

    │       └─ 二次发酵 ──→ 燃料乙醇 40%+（主产物 · 机车 / 发电机）
    │
    ├─ 输出层 ──→ 边境机甲 · 营地 · 酿造帮垄断利润
    │
