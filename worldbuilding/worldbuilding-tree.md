## 二、锈水危机

### 2.0 污染起因总述（v8 定稿）

赭锈带**从未有过可直饮的淡水**。殖民前，星球水循环已呈「赭锈」本色：陆海盆地边缘持续渗出重金属卤液，大气冷凝水则常年夹带辐射尘。

殖民灾难不是「打穿了毒层」，而是**把两类先天带病的水源并网了**：

```
锈水成因链（官方主干叙事）
    │
    ├─ 【先天层】星球本底
    │       ├─ 陆海卤缘：盆地边缘天然卤渗出（化学毒 · 全年）
    │       ├─ 嗜盐生态：卤缘水体载入本土菌群（生物毒 · 季节性）
    │       └─ 弱磁环境：辐射尘沉降污染一切露天集水（放射 · 尘暴季）
    │
    ├─ 【殖民层】人为放大
    │       ├─ C.Y. 2–8：前哨为求水量，沿卤缘修建截取渠 → 卤缘渠网
    │       ├─ C.Y. 5–10：冷凝塔网扩张，卤缘卤流与冷凝水并入同一源网
    │       ├─ 误判：以为稀释配比可管理毒性，未做源流分级
    │       ├─ C.Y. 8–12：矿冶尾液、酵母泥渗漏经支渠回流源网（支线污染）
    │       └─ 结果：垦区总水源网全域「锈化」── 不是某一点爆了，是整张网脏了
    │
    └─ 【时间层】十三月放大
            └─ 陆海砂潮上涨 → 卤缘渗出峰值 + 赤潮暴发 + 辐射扬尘 → 全年最毒
```

### 2.1 三层污染（官方主干 A + B + 辐射尘）

```
锈水来源（官方主干）
    │
    ├─ 【化学层】方案 A · 陆海卤缘渗出 ── 全年基线
    │       ├─ 盆地边缘天然卤化物涌出（高氯酸盐 / 砷 / 铜）
    │       └─ 殖民卤缘渠网集中收集 → 毒性随管网扩散至全域
    │
    ├─ 【生物层】方案 B · 流浪赤潮 ── Undecimber 峰值
    │       ├─ 嗜盐菌群沿卤缘与源网繁殖
    │       └─ 代谢产神经毒素 · 水呈铁锈红浊
    │
    └─ 【放射层】辐射尘 ── 尘暴季 / 冷凝塔 / 沙海扬尘
            ├─ 弱磁层无法阻挡双恒星风与轨道衰变尘
            ├─ 冷凝塔叶面与露天蓄源池直接接尘
            ├─ 沙海翻涌与沙豚集群迁徙 → 扬尘脉冲式污染
            └─ 三层叠加 → 水质降至「不可接触」级
```

| 层级 | 方案 | 污染机制 | 周期 |
|------|------|----------|------|
| 化学 | **A 陆海卤缘** | 盆地边缘天然卤渗出 + 卤缘渠网集中收集扩散 | 全年基线 |
| 生物 | **B 流浪赤潮** | 嗜盐菌群载入卤缘/源网水体，代谢产神经毒素 | **Undecimber 高峰** |
| 放射 | **辐射尘** | 放射性微粒污染冷凝塔、蓄源池与卤缘扬尘 | 尘暴季 + Undecimber 叠加 |

> **日常：** 卤缘化学毒已令源网水不可直饮。  
> **Undecimber：** 卤潮涨幅 + 赤潮 + 辐射扬尘峰值叠加。  
> **生态酵母：** 唯一可规模化的三重净化应答（见第三节）。

## 三、技术应答

### 3.1 生态酵母谱系

```
生态酵母（基础株 · Saccharomyces vitae）
    │
    ├─ W-Strain ──→ 源滤计划 · 滤兵（第一代植入者）
    │                   ├─ 锈水 → 内源生态酿（2–4%）
    │                   └─ 遗留：源醒（Source-Awake）· W-Strain 残留
    │
    ├─ R-Strain ──→ 民用救灾 · 生态酿（2–4%，公开合法）
    │
    └─ F-Strain ──→ 活燃料计划 · 醇民（Gen-0，已基本灭绝）
                        ▼ C.Y. 23 铁瓶计划
                    铁瓶（Gen-1）── ▼ C.Y. 37+ ──→ 酿民（Gen-2）
                        ├─ 瓶醒（Synth-Awake）
                        └─ 源醒（Source-Awake）── W-Strain 残留 / 幸存滤兵
```

### 3.2 燃料永续闭环

```
锈水 + 玉米浆 + 仙人掌糖浆 → 饲料搬运者 → 铁瓶 Gen-1
    ├─ 一次发酵 → 生态酿 2–4%
    └─ 二次发酵 → 燃料乙醇 40%+ → 机甲 / 营地 / 酿造帮利润
            └─ C.Y. 37+ 酿民觉醒 → 闭环裂缝
```
