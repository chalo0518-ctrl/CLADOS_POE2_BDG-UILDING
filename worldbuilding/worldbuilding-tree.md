## 叙事边界（定稿）

**全部故事只发生在赭锈带。** 地球及旧日国家势力仅存在于移民史、口头传说与破损档案中，**不作为活跃叙事线**——不写母星回流、跨星系通讯、宗主国复辟、地球视角章节或任何「地球上发生了什么」。

荒原叙事完全自治：势力、经济、伦理冲突均在本地生成。

## 四、三批移民史

```
移民史
    │
    ├─【第一批】国家资源竞赛 · 人类（C.Y. 0–8）
    │       │
    │       ├─ 多国争夺赭锈带
    │       ├─ 生态酵母植入首批移民胃中
    │       ├─ 使其能饮用锈水生存
    │       ├─ 内部文件暴露：嗜盐菌 0.05% 存活率
    │       ├─ 长期饮用 → 精神畸变
    │       ├─ 叠加长期辐射尘
    │       │       └─ 畸变为特殊生物磁场个体
    │       │               └─ 漂沦者 The Wander
    │       └─ 人类移民路线失败 · 国家撤回
    │
    ├─【第二批】Android 拓荒（C.Y. 8–25）
    │       │
    │       ├─ 模拟人群改造地表环境
    │       ├─ 植入生态酵母的 Android
    │       │       └─ 真正意义上的一代酿民 Gen-1
    │       ├─ 铁瓶 Synth-Rig
    │       │       ├─ 地表行驶的大型改造/酿运载具
    │       │       ├─ 一代酿民驻留其中
    │       │       └─ 形成 Neo-Western 铁瓶驿道
    │       ├─ 伴生铁瓶的 Android
    │       │       └─ 二代酿民 Gen-2
    │       └─ 二代中出现自主意识个体
    │               └─ 选择迁出铁瓶 · 逃亡
    │
    ├─【第三批】囚犯放逐（C.Y. 25–42）
    │       │
    │       ├─ 签署「自愿协议」的囚犯放逐拓荒
    │       ├─ 抵消旧日罪籍（档案用语 · 母星已无从追究）
    │       ├─ 水资源始终无法解决
    │       ├─ 殖民势力整体撤离 · 星系荒废
    │       ├─ 囚犯发现：酿民体内酵母发酵锈水
    │       │       ├─ 2–4% 琥珀色低度酒
    │       │       └─ 取祖先口述泰拉旧名「威士忌 Whiskey」
    │       ├─ 可直接饮用 · 或提取酵母再发酵
    │       ├─ 酵母随时间衰亡 · 须反复从酿民提取
    │       └─ 演化出赏金猎人 The Hunter · 猎杀酿民
    │
    └─【当前 C.Y. 42 Undecimber】
            ├─ 铁瓶车队仍行驶
            ├─ 二代觉醒酿民逃亡
            ├─ 赏金猎人追猎
            └─ 漂沦者游荡
```
