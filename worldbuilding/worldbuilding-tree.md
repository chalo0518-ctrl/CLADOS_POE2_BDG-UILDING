## 〇-b、水源污染起源 · 可选方案

生态酵母仍是所有方案共同的**技术应答**——差异只在「毒水从何而来」。

| 方案 | 名称 | 污染机制 | 叙事优势 | 与 Undecimber 的衔接 |
|------|------|----------|----------|----------------------|
| **A** | **深井毒卤（推荐主干）** | 殖民深井穿透含**高氯酸盐、砷、铜**的地下卤层；采出水全域不可直饮 | 硬核、符合外星地质；井与瓶意象天然绑定 | Undecimber 地下压力变化 → 卤层渗透率年度峰值 |
| **B** | **流浪赤潮（推荐叠加）** | 本土**嗜盐菌群**随季节风载入所有开放水源，代谢产生神经毒素 | 有生命感的「外星生态报复」；视觉化强（红浊水） | **赤潮高峰固定在 Undecimber**——整月水质最差 |
| **C** | **terraform 遗产** | 早期大气改造站泄漏催化剂，残留于冷凝水收集网 | 解释「谁该负责」：第一代殖民者遗产 | 改造站年度检修排废在 Undecimber 进行 |
| **D** | **矿冶尾液** | 为机甲燃料炼矿，尾液渗入含水层；财团隐瞒报告 | 直接对接酿造帮商业罪恶 | 财年结算在 Undecimber，排污量最大 |
| **E** | **边境化武残留** | 殖民内战 / 镇压本土势力时使用化学剂，污染流域 | 军事起源与井兵设定共振 | 战争结束纪念日恰逢 Undecimber |
| **F** | **尘暴带电粒子** | 双恒星尘暴携带带电金属微粒，降雨即毒 | 西部视觉极致；机车、铁瓶、尘暴同画面 | Undecimber 尘暴季，天空呈铁锈色 |

### 推荐组合（官方主干建议）

> **A 深井毒卤 + B 流浪赤潮**
>
> 殖民日常用水已因地质卤层而危险；每到 **Undecimber**，赤潮菌叠加涌入，水质降至年度最低——生态酵母需求最迫切、井源计划立项、Saloon 配给价最高、酿民觉醒传闻最多的月份。

### 生态酵母在各方案中的统一角色（保留）

无论污染起源为何，科学团均研发出 **生态酵母（Saccharomyces vitae）**：

- 以污染水为底物发酵
- 代谢分解 / 钝化毒素与重金属
- 产出 **2%–4% 生态酿**（含电解质与水分，可维持生命）
- 经二次发酵可提浓至 **40%+ 燃料乙醇**

**殖民宣传语：**「毒水变活路。」  
**未说出口的后半句：**「同一条菌链，也能把你变成油路。」

## 〇-c、十三月 Undecimber

```
垦历（C.Y.）
    │
    ├─ 一月 – 十二月（标准行政月 · 地球名沿用）
    │
    └─ 十三月 Undecimber（闰余月 · 37 垦日）
            │
            ├─ 天文成因
            │       └─ 垦锈带公转余数不被十二整除 → 殖民议会增设第 13 月
            │
            ├─ 社会语义
            │       ├─ 账外之月：地球母公司不计入 KPI 与编制
            │       ├─ 法外之月：悬赏翻倍、合同作废旧账、黑市开市
            │       └─ 禁忌之月：戒酒教称「上帝遗漏的月份」
            │
            ├─ 水文峰值（A+B 方案）
            │       ├─ 深井卤压上升 + 赤潮菌全面暴发
            │       ├─ 全年水质最差 · 生态酿配给价最高
            │       └─ 野生铁瓶多迁徙至赤潮边缘（一次发酵最活跃）
            │
            ├─ 技术史锚点
            │       ├─ C.Y. 12 Undecimber：生态酵母首次现场验证
            │       ├─ C.Y. 12 Undecimber：井源计划首批士兵植入
            │       ├─ C.Y. 23 Undecimber：铁瓶计划招标截止
            │       └─ C.Y. 42 Undecimber：二次发酵阈值临界（当前）
            │
            └─ 叙事惯例
                    ├─ Saloon 整月不打烊 · 酒保换班如换命
                    ├─ 赏金猎人：「Undecimber 的账，四月前不清。」
                    ├─ 酿造帮：年度配额在十三月 1 日重置
                    └─ 流浪酒帮：「在第十三月里，没有瓶子。」
```

**词源注：** *Undecimber* 借拉丁月份造词传统（*December* 原意为「第十月」），故意错置为「第十三月的异名」——殖民区自造的时间，母星不承认，正如铁瓶里的意识。
