### 派系关系速查（含戒酒教 · v18）

| 势力 | 地盘 | 核心利益 | 对酿民 | 对锈钉 |
|------|------|----------|--------|--------|
| **城酿署** | 城带 | 登记秩序 · 合法采酿 | 保护登记者 · 追捕未登记者 | 无关（他不进城） |
| **琥珀交易所** | 城带 | 威士忌定价 · 货币 | 商品化产出 | 无关 |
| **精品窖行** | 城带/灰市 | 窖酿垄断 | 签约即资产 · 脱约者送酿造帮或上悬赏 | 可暗中加价红标悬赏 |

| **铁瓶生态群** | 驿道 | 移动酿场 · 地表改造 | 一代驻内 · 伴生二代维护 | 中立通行 |
| **Saloon黑市** | 荒原 | 回收槽 · 出货 | 不问来路酵管 | 锈钉出货端 |
| **戒酒教·摒弃派** | 净泉聚落 | 用泉不用窖 | 敌对 · 隔离/告密交警署 | 可付情报借刀 |
| **戒酒教·圣火派** | 尘骑 | 秽须焚 | 敌对 · 焚杀净化 | 必杀 |
| **逃亡酿民网络** | 地下 | 互助逃生 | 自救 | 被悬赏/提取的对象 |
| **漂沦者** | 尘暴边缘 | 游荡生存 | 不适用（非酿民） | 警署高危悬赏 · 锈钉禁忌 |

        ▲                    │                      │

   精品窖行 ──脱约收编──      └────逃亡────► 缉拿手（锈钉）──提取──► Saloon黑市
                                                    ▲

## 七点五、戒酒教（Temperance Riders）

> 起源于生态酵母宗教化年代。教团**与酿民敌对**——视一代/二代酿民、铁瓶驻留者、城带登记酿民皆为**秽种**（行走酒窖），非人之容器。  
> C.Y. 28–32 城带合法采酿确立后，教团在「如何处置秽种」上分裂为**左翼温和派**与**右翼极端派**。

```
戒酒教残存（C.Y. 42）
    │
    ├─ 共同教义
    │       ├─ **酿民即敌** — 不庇护、不赎、不共存 · 与逃亡酿民网络对立
    │       ├─ 禁饮体内威士忌（劣酿/常酿/窖酿皆拒）
    │       ├─ 承认生态酵母可驯服锈水 · **拒绝**酵母驻于酿民之身
    │       └─ 警署悬赏非法提取：厌恶但可利用（秽种落缉拿手 · 不算圣事）
    │
    ├─ 完全摒弃派 The Total Rejection（左翼 · 温和派）
    │       │
    │       ├─ 核心教义：**用泉不用窖**
    │       │       ├─ **利用资源** — 取用锈水、离体生态酵母、去毒工艺 · 不取自酿民体内
    │       │       ├─ **完全摒弃** — 摒弃一切酿民：不接触、不交易、不承认其「人」格
    │       │       └─ 城带登记酿民亦秽种 · 不因合法而赦免
    │       │
    │       ├─ 对酿民手段（非火）
    │       │       ├─ **隔离带** — 聚落外围拒酿民入境 · 污名化铁瓶线
    │       │       ├─ **诱捕移交** — 追踪逃亡/未登记酿民 · 交警署或城酿署（借刀）
    │       │       ├─ **断供** — 抵制含酵产物 · 打击 Saloon 黑市需求侧
    │       │       └─ **净泉工艺** — 只售去毒无醇锈水 · 证明「无需酿民也能活」
    │       │
    │       ├─ 生存方式
    │       │       ├─ **净泉聚落** — 城缘/荒原边缘 · 离体酵母培菌池（非人身）
    │       │       ├─ **摒弃商路** — 卖洁净锈水、滤材、去毒服务 · 换粮食弹药
    │       │       ├─ **告密网络** — 向警署/城酿署出卖酿民坐标 · 换保护或赏格
    │       │       └─ **铁瓶线抵制** — 不攻击群落 · 但拒绝通行、拒修、拒贸易
    │       │
    │       └─ 与当代关系
    │               ├─ 对锈钉：不焚杀 · 但愿付情报费引其接酿民悬赏
    │               ├─ 对城带：游说限制酿民配额 · 务实合作人类当局
    │               └─ 斥圣火派：嗜杀败坏「摒弃」之道
    │
    └─ 圣火派 The Sacred Fire（右翼 · 极端派）
            │
            ├─ 核心教义：**秽须焚**
            │       ├─ 酿民不可移交、不可利用 · **须以圣火净化**
            │       ├─ 体内威士忌、发酵舱、回收槽酵管 — 同罪同焚
            │       └─ 铁瓶生态群即移动祭坛 · 外露发酵舱优先焚毁
            │
            ├─ 对酿民手段（火与杀）
            │       ├─ **圣火净化** — 焚杀酿民 · 毁窖芯 · 不留复提取可能
            │       ├─ **净火行动** — 焚 Saloon 回收槽、黑市酵舱、铁瓶外露舱
            │       ├─ **断酿伏击** — 袭击警署交货点/缉拿手押解队 · 先杀酿民再砸酵管
            │       └─ **铁瓶线圣战** — 十三月尘暴掩护 · 突袭群落边缘单元
            │
            ├─ 生存方式
            │       ├─ **尘骑巡游** — 机动武装 · 少设固定聚落
            │       ├─ **殉火队** — 自杀式焚舱 · 以火证明信仰
            │       ├─ **劫火补给** — 掠袭含酵物资后焚烧 · 不囤积威士忌
            │       └─ **净居余众** — 荒漠深处只饮去毒无醇锈水 · 与完全摒弃派断绝往来
            │
            └─ 与当代关系
                    ├─ 对锈钉：**必杀** — 提取手与秽种同焚
                    ├─ 对警署：敌 · 断酿伏击不打招呼 · **圣火派成员亦上警署悬赏**
                    ├─ 对城带：恐袭琥珀交易所外缘 · 城酿署最高危宗教武装
                    └─ 斥完全摒弃派：与秽种交易情报 · 等同亵渎
```

### 两派对照（写作速查）

| | **完全摒弃派（左翼·温和）** | **圣火派（右翼·极端）** |
|---|---------------------------|------------------------|
| **核心句** | 用泉不用窖 | 秽须焚 |
| **对酿民** | 敌对 · 隔离/诱捕/移交 · **不亲手焚杀** | 敌对 · **焚杀净化** · 不留活口 |
| **生态酵母** | **利用**离体培菌去毒锈水 | 可用去毒锈水 · 恨的是「窖」不是泉 |
| **城带登记酿民** | 仍视为秽种 · 游说限制 | 一律焚净目标 |
| **锈钉/缉拿手** | 厌恶 · 可买情报借刀 | 必杀 · 断酿伏击 |
| **铁瓶生态群** | 抵制不通航 · 不焚群落 | 圣战焚外露发酵舱 |
| **人数与气质** | 较多 · 冷峻务实 | 较少 · 狂热暴力 |

```
    酿民 ◄────敌对────► 戒酒教（两派共识）
              │
      ┌───────┴───────┐
  完全摒弃派      圣火派
  （隔离/借刀）   （焚杀/圣战）
```

## 七点六、当地警署与悬赏体系（Constabulary Post）

> C.Y. 30 前后，荒原「猎人公会」随第三批移民秩序瓦解而裁撤；悬赏接取统归**当地警署**——城带设分局，荒原驿道设驿站驻点。布告栏写的是治安，荒原执行的是生存。

```
当地警署（C.Y. 42）
    │
    ├─ 职能
    │       ├─ **悬赏布告** — 城缘/驿站驻点挂牌 · 登记缉拿手接取
    │       ├─ **抓捕凶犯** — 谋杀、劫酵、焚舱、袭警等刑事案件
    │       ├─ **追捕潜逃奴役酿民** — 酿造帮园区出逃 · 城酿署协查脱约者
    │       └─ **羁押移交** — 活捉目标押解城带分局或归还酿造帮园区
    │
    ├─ 悬赏目标四类（布告栏覆盖）
    │       │
    │       ├─ **酿民类**
    │       │       ├─ 潜逃奴役酿民（酿造帮签约出逃）
    │       │       ├─ 未登记/逃亡酿民
    │       │       └─ 脱约精品酿民（窖行/酿造帮暗中推动上榜）
    │       │
    │       ├─ **圣火派**（戒酒教净化武装）
    │       │       ├─ 净火行动 · 焚舱恐袭 · 袭警
    │       │       └─ 城酿署令：宗教武装 · 原则上格杀勿论
    │       │
    │       ├─ **凶犯类**
    │       │       ├─ 谋杀犯
    │       │       ├─ 劫酵/劫火案犯
    │       │       └─ 袭警 · 劫掠酿造帮园区
    │       │
    │       └─ **高危群体：漂沦者**
    │               ├─ 城酿署令：精神畸变 · 生物磁场干扰导航
    │               ├─ 原则上活捉隔离 · 实际荒原多按「格杀许可」执行
    │               └─ 锈钉个人禁忌：不接漂沦者悬赏（磁场干扰提取器）
    │
    ├─ 缉拿手制度
    │       ├─ 在警署登记 · 押保证金 · 按悬赏类别接活
    │       ├─ **名义**：抓捕 · 押解 · 领取赏格
    │       ├─ **荒原常态**：对酿民目标先击晕非法提取 → 再交人/交尸/交部分窖芯
    │       └─ 警署睁一只眼闭一只眼——酿民类悬赏酬金低，黑市酵价补差额
    │
    └─ 与城带关系
            ├─ 城带分局签发红标（高酬 · 精品酿民 · 加密委托人）
            ├─ 驿站驻点同步布告 · 锈钉在城缘 Saloon 旁警署挂牌
            └─ 城酿署脚注：「与登记酿民冲突以城酿署登记为准」——多数缉拿手不读
```

| 悬赏类别 | 典型目标 | 警署口径 | 荒原执行实况 |
|----------|----------|----------|--------------|
| 酿民类 | 出逃奴役酿民、脱约精品酿民 | 追捕归案 | 击晕提取 → 黑市出手 → 交人/假交 |
| 圣火派 | 尘骑、殉火队 | 宗教武装缉拿 | 格杀 · 少交活口 |
| 凶犯类 | 谋杀犯、劫酵犯 | 刑事缉拿 | 按布告活捉/格杀 |
| 漂沦者 | 尘暴边缘游荡者 | 高危隔离 | 多数猎队格杀 · 锈钉不接 |

## 七点七、酿造帮（Brewing Syndicate）

> **本质是高利贷与合同奴役集团，不是人权组织。** 专盯负债、脱约、未登记酿民，以「安置就业」「偿还积欠」为名诓签卖身条款，将其锁入围栏**园区**充当酿造工具。

```
酿造帮（C.Y. 42）
    │
    ├─ 起源
    │       ├─ 第三批移民后荒原金融真空 · 高利贷填补
    │       ├─ 城缘人类商户、窖行灰市代理人出资
    │       └─ 与城酿署合法登记并行 · 专吃「进不了城」的酿民
    │
    ├─ 运作模式
    │       ├─ **诱贷** — 锈水滤材、酵苗、医疗、驿道通行费 → 复利滚欠
    │       ├─ **卖身条款** — 不合理条框：终身酿造义务、出逃即全债加速、窖芯抵押
    │       ├─ **园区囚禁** — 围栏发酵舱 · 24 垦时排班 · 体内采酿直接抽走
    │       ├─ **精品筛选** — 窖酿潜质者加价转卖窖行灰市
    │       └─ **出逃悬赏** — 向当地警署挂「潜逃奴役酿民」· 酬金由园区出
    │
    ├─ 园区生态
    │       ├─ 城缘外环、铁瓶线岔口 · 武装看守
    │       ├─ 酿民编号管理（如 R-441）· 与警署悬赏系统打通
    │       ├─ 伴生未觉醒二代有时被整批「抵债」收入
    │       └─ 死亡/报废：窖芯拆解 · 不留档案
    │
    ├─ 对外形象
    │       ├─ 宣传：「给酿民一口饭吃」「比荒原猎杀人道」
    │       ├─ 实际：债务陷阱 · 无退出机制 · 身体即抵押品
    │       └─ 精品窖行暗中收编脱约者时，常经酿造帮「洗合同」
    │
    └─ 与当代势力
            ├─ 对城酿署：表面合法劳务合同 · 实则城酿署懒得管荒原债奴
            ├─ 对警署：出逃即悬赏 · 警署赚接办费 · 酿造帮赚威慑
            ├─ 对戒酒教摒弃派：偶有合作（告密出逃酿民坐标）
            ├─ 对戒酒教圣火派：敌对 · 圣火派视园区为「秽窖」欲焚
            ├─ 对铁瓶生态群：诱拐落单伴生酿民 · 不惹整群
            └─ 对锈钉：稳定赏金来源 · 红标精品常是「园区出逃」或「窖行转挂」
```

| | **城带登记酿民** | **酿造帮园区酿民** |
|---|-----------------|-------------------|
| **身份** | 城酿署保护 · 合同采酿 | 债务奴役 · 卖身条款 |
| **产出归属** | 换琥珀币 · 配额制 | 帮会抽走 · 抵债无休 |
| **出逃后果** | 脱约 · 窖行/警署悬赏 | 潜逃奴役酿民 · 警署标准悬赏 |
| **叙事功能** | 合法价码 · 窖七可对标的「人」 | 奴役真相 · R-441 等编号来源 |
