## 〇-b、水源污染 · 官方主干（A + B + 辐射尘）

### 污染三层结构（已定稿）

```
毒水来源（官方主干）
    │
    ├─ 【化学层】方案 A · 深井毒卤 ── 全年
    │       └─ 深井穿透卤层 → 高氯酸盐 / 砷 / 铜
    │
    ├─ 【生物层】方案 B · 流浪赤潮 ── Undecimber 峰值
    │       └─ 嗜盐菌群 → 神经毒素 · 水呈铁锈红浊
    │
    └─ 【放射层】辐射尘 ── 尘暴季 / 沙海扬尘 · Undecimber 叠加
            │
            ├─ 弱磁层无法阻挡双恒星风与轨道衰变尘
            ├─ 辐射尘沉降 → 渗入重砂层 → 随卤水上涌进入深井
            ├─ 沙海翻涌与沙豚集群迁徙 → 扬尘脉冲式污染
            └─ 三层叠加时，水质降至「不可接触」级
```

| 层级 | 方案 | 污染机制 | 周期 |
|------|------|----------|------|
| 化学 | **A 深井毒卤** | 殖民深井穿透地下卤层，采出水含重金属盐 | 全年基线 |
| 生物 | **B 流浪赤潮** | 嗜盐菌群载入水源，代谢产神经毒素 | **Undecimber 高峰** |
| 放射 | **辐射尘** | 放射性微粒随尘暴与沙海扬尘进入水源 | 尘暴季 + Undecimber 叠加 |

> **日常：** 化学卤毒已令水不可直饮。  
> **Undecimber：** 赤潮 + 辐射尘扬尘峰值叠加，全年最毒月份。  
> **生态酵母：** 唯一可规模化的三重净化应答。

### 其他方案（C–F）的地位

方案 C–F（terraform 遗产 / 矿冶尾液 / 化武残留 / 尘暴带电粒子）可作为**局部支线或历史诱因**，不再作为主行星污染主干；其中 **F 尘暴带电粒子** 可视作辐射尘机制的民间误传版本。

### 生态酵母的统一角色（保留并扩展）

**生态酵母（*Saccharomyces vitae*）** 针对 A+B+辐射尘三重污染设计：

- 以污染水为底物发酵
- 代谢分解 / 钝化化学毒素与重金属
- 钝化 / 包裹放射性核素，沉降为发酵渣（「酵母泥」需专门处置）
- 产出 **2%–4% 生态酿**（含电解质与水分，可维持生命）
- 经二次发酵可提浓至 **40%+ 燃料乙醇**

**殖民宣传语：**「毒水变活路。」  
**回收帮私下：**「酵母泥比毒水更麻烦。」

```
垦历（C.Y.）

            │
            ├─ 水文峰值（A + B + 辐射尘）
            │       ├─ 深井卤压上升 + 赤潮菌全面暴发 + 沙海辐射扬尘
            │       ├─ 全年水质最差 · 生态酿配给价最高
            │       └─ 野生铁瓶迁徙至赤潮边缘；沙豚集群穿越沙海

            │       ├─ C.Y. 12 Undecimber：井源计划首批士兵植入
            │       ├─ C.Y. 23 Undecimber：铁瓶计划招标截止

                    ├─ Saloon 整月不打烊 · 酒保换班如换命
```

## 〇-d、陆海 · 沙海（The Landsea）

```
垦锈带地表
    │
    ├─ 固陆（Rock Rim）── 城镇 · 深井 · 牧场 · 驿道
    │
    └─ 陆海 / 沙海（The Landsea）
            │
            ├─ 物质：重砂（Dense Sand）
            │       ├─ 硅-铁复合微粒 · 密度约为地球沙漠沙的 2.8–3.2 倍
            │       ├─ 风力与地热震动下呈现缓慢「流态」——像海，却是陆地
            │       └─ 辐射尘大量沉积于重砂层，翻涌时扬尘入云
            │
            ├─ 地理
            │       ├─ 环绕垦区盆地的铁锈色「砂洋」 · 占可殖民地表约 40%
            │       ├─ 砂波（Sand Swell）替代海浪 · 砂啸（Sand Roar）替代风暴潮
            │       └─ 多数深井卤层出露点位于陆海盆地边缘
            │
            ├─ 殖民利用
            │       ├─ 饲料搬运者沿砂岸驿道行驶（不入沙海深处）
            │       ├─ 野生铁瓶随砂流漂移 · 酿造帮牧场多建在陆海边缘
            │       ├─ 辐射尘富集区 · 酵母泥处置场多秘密设于沙海下游
            │       └─ 赏金猎人称陆海深处为「无井地带」
            │
            └─ 与毒水闭环
                    ├─ 重砂层储存辐射尘 → 卤水上涌时带入深井
                    ├─ 赤潮菌孢在砂表休眠 → Undecimber 随风入水
                    └─ 生态酵母渣（含钝化辐射核素）→ 常非法倾倒入沙海
```

**视觉意象：** 铁锈色的沙浪在无云天空下起伏；机甲与商队像行船一样沿「海岸线」行进，而深处有东西在砂下游泳。

## 〇-e、沙豚（Sand Porpoise / *Magnetodelphis arenarius*）

```
沙海生态
    │
    └─ 沙豚（Sand Porpoise）
            │
            ├─ 本质
            │       ├─ 本土大型哺乳动物 · 体长 3–6 米
            │       ├─ 颅骨两侧具生物电磁器官（磁泳腺）
            │       └─ 释放局域磁场 → 降低周围重砂有效密度与摩擦
            │
            ├─ 能力：磁泳（Magneto-Swimming）
            │       ├─ 在陆海重砂中像豚类游水一样前进
            │       ├─ 集群迁徙时翻起砂浪，释放辐射尘与赤潮孢
            │       └─ 民间谚语：「沙豚起波，井水变浊。」
            │
            ├─ 与殖民势力的关系
            │       ├─ 酿造帮：猎杀取磁泳腺（铁瓶导航 / 军用研究）
            │       ├─ 流浪酒帮：部分派系视沙豚为「未容器化的净生者」
            │       ├─ 戒酒教·收割派：称沙豚为「会游泳的清泉使者」
            │       ├─ 戒酒教·纯净派：视为应焚毁的「脏物载体」
            │       └─ 边境部落：以沙豚迁徙预测 Undecimber 毒水峰值
            │
            ├─ 与井 / 瓶 对照
            │       ├─ 井兵：人体内净化毒水
            │       ├─ 铁瓶：机械容器产燃料
            │       └─ 沙豚：自然进化 · 不产酒 · 磁泳穿越毒砂——第四镜像
            │
            └─ 叙事资源
                    ├─ 沙豚骑士（驯化尝试 · 成功率极低）
                    ├─ 砂下救援 / 砂下伏击
                    ├─ 磁泳腺黑市（与酿民接口抽取并列的器官贸易）
                    └─ C.Y. 42 Undecimber：异常大规模集群迁徙 → 觉醒催化剂？
```

**设定句：** 人类把自身做成瓶子产酒；沙豚生来就能在毒砂里游泳——从未被问过愿不愿成为解决方案。
